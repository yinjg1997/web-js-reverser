就目前来讲，JSVMP 的逆向方法有三种（自动化不算）：RPC
远程调用，补环境，日志断点还原算法，其中日志断点也称为插桩，找到关键位置，输出关键参数的日志信息，从结果往上倒推生成逻辑，以达到算法还原的目的

超长编码字符串就是被保护的原始 JS 逻辑编译后的字节码

整个外层 IIFE 是 JSVMP 系统，内部有明确分工：
